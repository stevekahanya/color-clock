# React Dynamic Color Clock

A productivity dashboard widget that displays the current date and time.

## Technologies Used
- **React**: Frontend framework.
- **Vite**: Build tool for fast development.
- **date-fns**: For precise date formatting.

## How to Run Locally
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the application.
