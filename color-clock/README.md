# Color Clock Widget

A dynamic, colorful digital clock built with React, Vite, and date-fns.

## Features
- Real-time ticking clock (updates every second).
- Formatted date and time using `date-fns`.
- Responsive, vibrant CSS design with glassmorphism effects.
